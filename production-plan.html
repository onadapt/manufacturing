Planning Assumptions

Lot size 1 drone prototype

Scale by multiplying labor and material pull quantities by the planned build quantity.

Material control Kit before build

All serialized and safety-sensitive parts are staged before Airframe starts.

Quality model Gate by station

Each station releases only accepted work to the next operation.

Rework path Controlled return

Failed QA or test items route through Rework before returning to Kitting or the failed station.

Other Stuff To Define

Labor standard Direct + support

Track direct labor by station and add indirect time for material handling, QA review, and rework disposition.

Material release Shortage visible

Before a build starts, the kit should show available, short, substituted, and serialized parts.

Standard script Operator prompts

Each station needs a script with setup, action steps, hold points, pass/fail checks, and signoff.

Production records Traceable build

Capture serial number, firmware version, inspection results, rework history, and final inventory location.
